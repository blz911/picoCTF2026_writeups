# picoCTF 2026 Writeups 🚩

My journey through the **picoCTF 2026** challenges. I’ve documented every step, from initial recon to the final flag, with the goal of making complex security concepts easier to understand.

## 🛠️ Toolkit & Tech Stack
I utilize a variety of industry-standard tools to analyze and exploit these challenges:

* **Web Exploitation:** Burp Suite, Browser DevTools (Console/Network/Debugger)
* **Cryptography:** Python (`pycryptodome`), SageMath, FactorDB (for RSA attacks)
* **General Purpose:** CyberChef, Bash scripting, Python

## 📂 Repository Structure
The writeups are organized by category:
* **/Cryptography** - RSA, AES, and Number Theory challenges.
* **/Web-Exploitation** - SQLi, XSS, and Logic Flaws.
* **/Forensics** - Metadata analysis and file carving.
* **/Reverse-Engineering** - Binary analysis and decompilation.
